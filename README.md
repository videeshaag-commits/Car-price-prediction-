# 🚗 Car Price Prediction (AI/ML Project)

This project uses Machine Learning to predict car prices based on features like year, kilometers driven, and fuel type.

## 🔧 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn

## 📊 Features
- Data preprocessing (handling categorical variables)
- Train-test split
- Linear Regression model
- Model evaluation using MAE and R2 Score

## 📈 Results
- Achieved good prediction accuracy on sample dataset
- Evaluated using standard ML metrics

## ▶️ How to Run
1. Install dependencies:
   pip install pandas numpy scikit-learn

2. Run the file:
   python car_price_prediction.py
