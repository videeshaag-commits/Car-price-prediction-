🚗 Car Price Prediction (AI/ML Project)

This project uses Machine Learning to predict car prices based on features like year, kilometers driven, and fuel type.

---

🔧 Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn

---

📊 Features

- Data preprocessing (handling categorical variables)
- Train-test split
- Linear Regression model
- Model evaluation using MAE and R² Score

---

📂 Dataset

- Sample dataset used for demonstration
- Can be replaced with real-world datasets (e.g., Kaggle)

---

📈 Results

- Evaluated model performance using MAE and R² score
- Provides reasonable predictions based on input features

---

📌 Example Output

Predicted Price: 7.8 (approx)

---

🧠 Skills Gained

- Machine Learning fundamentals
- Data preprocessing techniques
- Model training and evaluation

---

▶️ How to Run

1. Install dependencies:
   pip install pandas numpy scikit-learn

2. Run the file:
   python car_price_prediction.py
